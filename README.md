# Employee Management API

## Technology Stack

- .NET 8 Web API
- Entity Framework Core
- SQL Server
- Repository Pattern
- Swagger
- JWT Authentication

## Features

- Add Employee
- Get Employee List
- Get Employee By Id
- Update Employee
- Delete Employee

## Run

1. Update connection string in appsettings.json

2. Run Package Manager Console:
   Add-Migration InitialCreate
   Update-Database
   
3. Run project

4. Open Swagger:
   https://localhost:5001/swagger
